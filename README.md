# TamTam Frontend Builder

## Use it
Implement the following node scripts in your project:

```
"scripts": {
    "gulp": "cd ./node_modules/tamtam-frontend-builder/bin/ && node start.js",
    "start": "cd ./node_modules/tamtam-frontend-builder/bin/ && node start.js",
    "build": "cd ./node_modules/tamtam-frontend-builder/bin/ && node build.js",
    "dist": "cd ./node_modules/tamtam-frontend-builder/bin/ && node dist.js",
    "task": "cd ./node_modules/tamtam-frontend-builder/bin/ && node task.js",
    "bamboo": "cd ./node_modules/tamtam-frontend-builder/bin/ && node deploy.js",
    "server": "cd ./node_modules/tamtam-frontend-builder/bin/ && node deploy.js"
  }
```
